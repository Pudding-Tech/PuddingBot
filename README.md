# PuddingBot - Discord bot

PuddingBot serves the Discord server Puddings.

### Environment variables

**Required:**  
> PUDDINGBOT_TOKEN  
> PUDDINGBOT_CLIENT_ID  
> GUILD_ID  

**Optional:**
> NODE_ENV  
> TZ  
> PLEX_CLIENT_IDENTIFIER  
> PUDDINGFLIX_IP  
> PUDDINGFLIX_PORT  
> PUDDINGFLIX_TOKEN  
> DUCKFLIX_IP  
> DUCKFLIX_PORT  
> DUCKFLIX_TOKEN
